<h3>Silver IV</h3> 1978 

```python
num = int(input())
nums = list(map(int, input().split()))
    
for i in range(num):
    if nums[i] == 1:
        num -= 1
    else:
        for j in range(2, nums[i]):
            if nums[i] % j == 0:
                num -= 1
                break
            
print(num)
```
