<h3> Silver IV</h3>  

#### 10845 큐
```python
import sys
n = int(sys.stdin.readline())

Q = []

for i in range(n):
    s = sys.stdin.readline().split()
    
    if s[0] =='push':
        Q.insert(0,s[1])
    
    elif s[0]== 'pop':
        if len(Q) != 0:
            print(Q.pop())
        else :
            print(-1)
    
    elif s[0] == 'size':
        print(len(Q))
    
    elif s[0] == 'empty':
        if len(Q) == 0 : 
            print(1)
        else :
            print(0)
    
    elif s[0] == 'front':
        if len(Q) == 0:
            print(-1)
        else:
            print(Q[-1])
            
    elif s[0] == 'back':
        if len(Q) == 0 :
           print(-1)
        else: 
            print(Q[0])
```
