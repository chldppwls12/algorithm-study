<h3> Silver V</h3>  7568 덩치 

```python
n = int(input())
li = []

for i in range (n):
    w,h = map(int,input().split())
    li.append((w,h))

for i in li:
    rank = 1
    for j in li:
        if i[0]!=j[0] and i[1]!=j[1]:
            if i[0]<j[0] and i[1]<j[1]:
                rank+=1
    print(rank,end=' ')
```


<h3> Silver V</h3>  14501 퇴사
 
```python
n = int(input())
workday=[]
Mpay = [0]*n

for i in range(n):
    workday.append(list(map(int,input().split())))
    
for i in range(n-1,-1,-1):
    day = workday[i][0]
    pay = workday[i][1]
    
    if day >n-i: #남은 기간보다 상담일이 긴 경우 
        if i !=n-1 :
            Mpay[i] = Mpay[i+1] #이전 최댓값 저장 
        continue
        
    if i == n-1:
        Mpay[i] = pay
    elif i+day ==n :
        Mpay[i] = max(pay,Mpay[i+1])
    else:
        Mpay[i] = max(pay + Mpay[i + day], Mpay[i+1])
    
print(Mpay[0])
``` 

<h3> Silver III</h3>  2579 계단오르기 
 
```python
n = int(input())
#점수 리스트 
s = [0 for i in range(301)]
#점수의 합 저장 리스트 
dp = [0 for i in range (301)]

for i in range(n):
    s[i] = int(input())
      
dp[0]=s[0]
dp[1]=s[0]+s[1]
dp[2]=max(s[1]+s[2],s[0]+s[2])

for i in range(3,n):
    dp[i]=max(dp[i-3]+s[i-1]+s[i],dp[i-2]+s[i])
print(dp[n-1])
```
