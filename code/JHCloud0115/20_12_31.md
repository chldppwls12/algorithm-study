<h3>Bronze I </h3>  

#### 4344 평균은 넘겠지
```python
import sys
n = int(input())
result = []
for i in range(n):
    num = list(map(int,sys.stdin.readline().split()))
    cnt = 0
    avg = sum(num[1:])/num[0]
    for a,b in enumerate(num):
        if num[a] > avg :
            cnt+=1
            result.append(cnt/num[0]*100)

for j in range(n):
  print(format(result[j],'.3f')+'%')
```
