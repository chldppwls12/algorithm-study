<h3>Silver IV</h3>  1002 터렛

```python
t = int(input())
for i in range (t) : 
    x1,y1,r1,x2,y2,r2 = map(int,input().split())
    d = (x1-x2)**2 + (y1-y2)**2
    rsum = (r1+r2)**2
    rsub = (r1-r2)**2
    if d == 0: #좌표 같을 때 
        if r1 == r2 : #원이 겹칠 때 
            print(-1)
        else : # 한 원이 다른 원 포함하는 경우 
            print(0)
    else : 
        if((d == rsum) or (d == rsub)): # 한점만 겹치는 경우
            print(1)
        elif((d < rsum) and (d > rsub)): # 겹치는 점이 2개인 경우
            print(2)
        else: # 아예 떨어져 있는 경우
            print(0)
```
<h3>Silver II</h3> 1003 피보나치  

```python
t = int(input())

for i in range(0,t):
    f =int(input())
    F = [0,1]
    if f == 0 :
        print(1,0)
    elif f ==1 : 
        print(0,1)
    elif f ==2 : 
        print(1,1)
    else : 
        for a in range(2,f+1):
            F.append(F[a-1]+F[a-2])
        print(F[f-1],F[f])

```

<h3>Silver II</h3> 1929 소수 구하기 

```python
#시간초과 
m,n = map(int,input().split())
def isPrime(a):
    if a >2 :
        for i in range(2,a):
            if(a%i==0):
                return False
    return True

for i in range (m,n+1):
    if isPrime(i):
        print(i) 
```
